# frontend-magang-webbiro
Berisi file HTML, JS, dan CSS untuk halaman website magang 
Proyek ini merupakan pengembangan frontend website Magang Biro Umum dan Pengadaan Barang dan Jasa, yang dirancang untuk mendukung proses pendaftaran, informasi, serta pengelolaan aktivitas magang secara lebih terstruktur dan mudah diakses. 
Website ini dibangun menggunakan standar pengembangan website dengan memanfaatkan HTML, CSS, dan JavaScript, serta dipadukan dengan komponen UI yang mendukung pengalaman user yang intuitif. Setiap halaman disusun sesuai dengan kebutuhan proses magang, mulai dari informasi umum, pendaftaran, login, hingga pengelolaan data oleh admin. 
Repositori ini dibuat sebagai dokumentasi sekaligus sarana kolaborasi dalam proses pengembangan sistem magang yang lebih profesional dan transparan di lingkungan Biro Umum dan Pengadaan Barang dan Jasa.
